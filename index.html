《Javascript 高等程序设计》第21章 Ajax 与 Comet

发表于 2016-04-07   |   分类于 Web前端   |  

IE6及更早期版本的写法不做考虑，上一个时代的东西就随时代而去吧:P

Ajax：Asynchronous JavaScript And XML ，其核心是XMLHttpRequest对象（XHR）
注：Ajax通信与数据格式无关，不一定是XML数据

1、XMLHttpRequest 1级

1.1、XHR用法：

  • 创建XHR：var xhr = new XMLHttpRequest();
  • 启动一个请求：xhr.open(“get”, “index.php”, false);
    • 参数为：发送请求类型、请求的URL、是否异步发送请求
    • 不会发送请求，而是准备发送
  • 发送请求：xhr.send();
    • 参数为：请求主体的数据，没有需传入null
  • 响应数据：响应数据会返回给xhr对象，属性为：
    • responseText：响应主体文本
    • responseXML：若响应内容类型是“text/xml”或“application/xml”，则包含响应数据的XML DOM 文档
    • status：响应的HTTP状态
    • statusText：HTTP状态说明
  • 同步请求：检查响应状态并做相应处理
