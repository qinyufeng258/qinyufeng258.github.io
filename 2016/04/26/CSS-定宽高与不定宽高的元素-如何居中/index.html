CSS 定宽高与不定宽高的元素 如何居中

发表于 2016-04-26   |   分类于 WEB前端   |  

网易17届春招实习面试被问到了这题，但是苦于平时积累一般，回答得有很多纰漏，特地整理一下。

水平居中：

1.满足定宽和block元素：对需要居中的元素应用margin:0 auto实现

2.满足inline元素或inline-*元素：对父元素应用text-align:center实现

3.满足不定宽和block元素的水平居中：

  • 加入table标签（包括tbody、tr、td），对table标签应用margin:0 auto

增加了无意义的语义化标签，复杂了嵌套深度

  • 设置display:inline | inline-block，对父元素应用text-align:center
    • 如果设定为inline元素，则无法设定宽高
    • 因此设定为inline-block相对比较好
  • 父元素设定position:relative定位，应用left:50%,子元素设定position:relative定位，应用left:-50%

父元素偏移50%后，子元素在父元素的基础上反向偏移父元素长度的一半，从而达到居中

但是设定position:relative会带来一定的副作用

  • 父元素设定flex定位
  • 父元素设定position:relative定位，子元素设定position:absolute定位，应用left:50%,translateX(-50%)

垂直居中：

1.满足定高和单行文本：设定父元素的line-height 和 height一致

2.满足定高和多行文本|图片|block元素：

  • 加入table标签（包括tbody、tr、td），对td标签应用height:xxxpx；

增加了无意义的语义化标签，复杂了嵌套深度

  • 使用position:absolute top:50%、margin-top:-XXpx负补偿进行居中

3.满足不定高和多行文本 | 图片 | block元素：

  • 对父元素设定display: table-cell; vertical-align: middle;

注意：设定margin/padding无效，需要对父元素的父元素使用

  • 父元素设定flex定位
  • 父元素设定position: relative定位，子元素设定position: absolute定位，应用top:50%,translateY(-50%);

水平垂直居中：

1.父元素设定flex布局

2.父元素设定table-cell

注意：设定margin/padding无效，需要对父元素的父元素使用

3.需要居中的元素外套一个空div元素，然后父元素应用position:relative，空div应用position:absolute left:50% top:50%，子元素用margin-left: -50% margin-top:-50%

效果：

4.父元素使用position:relative 子元素使用position:absolute定位，left:50% top50%，然后使用translate(-50%,-50%)

5.定宽定高的情况下使用margin定值负补偿即可

6.神器： margin:auto; margin: auto; position: absolute; top: 0; left: 0; bottom: 0; right: 0; 居中

缺陷：

  • 必须设定高度
  • 建议设置overflow:auto来防止内容越界溢出。（查看溢出Overflow）
  • 在Windows Phone设备上不起作用
